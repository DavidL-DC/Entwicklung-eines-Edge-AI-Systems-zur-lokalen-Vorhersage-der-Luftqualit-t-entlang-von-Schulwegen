# Entwicklung-eines-Edge-AI-Systems-zur-lokalen-Vorhersage-der-Luftqualit-t-entlang-von-Schulwegen

Entwicklung eines Edge-AI-Systems zur lokalen Vorhersage der Luftqualität entlang von Schulwegen von David Lohmann für den Kurs Projekt: Edge-AI DLBAIPEAI01_D im Studiengang B.Sc. Angewandte Künstliche Intelligenz

Installation:
1. Alle Dateien in einen Ordner legen
2. Aktuelle Python Version installieren
3. pip install -r requirements.txt ausführen, um Abhängigkeiten zu installieren
